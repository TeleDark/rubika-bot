دستوراتی که وقتی برای اولین بار به سروری که تازه ساختی کانکت میشی باید اجرا کنی :
apt update
apt install unrar -y
apt install python3-pip -y
apt install ffmpeg libsm6 libxext6 -y

بعدش یه پوشه میسازی با اسم دلخواه
mkdir "dir_name"
به جای dir_name اسم پوشه رو میزاری


بعد سورس کد رو اپلود میکنی تو پوشه


cd "dir_name"
به جای dir_name اسم پوشه ای که ساخته بودی رو میزاری


unrar x file.rar
به جای file.rar باید اسم فایل سورس کد رو بزاری هرچی که هست


pip install -r requirements.txt


خب کارایی که برای اولین بار باید انجام بدی تموم شد


حالا وقتی برای بار دوم یا هر چندم به سرور کانکت میشی و میخوای مثلا رباتو روشن خاموش کنی یا لاگین کنی اینکارو میکنی :


اول باید وارد پوشه ای که سورس کد رباتو توش اپلود کردی  بشی با این دستور :
cd "dir_name"
به جای dir_name اسم پوشه

برای لاگین کردن : python3 login.py

برای اجرا : python3 start.py

برای استوپ کردن : python3 stop.py
